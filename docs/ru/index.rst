Stof (Selenium Tests Organization Framework) -- компактный фреймворк для организации кода автоматизированных тестов,
написанных на языке CoffeeScript с использованием Selenium Webdriver. Он определяет правила организации файлов
тест-спецификаций, предоставляет базовые классы для паттернов тестирования (в частности, PageObject), а также
добавляет синтаксический сахар для удобного доступа к вспомогательным классам и командам веб-драйвера.

Основная задача, которая ставилась при разработке фреймворка -- максимально уменьшить количество вспомогательного кода
(не относящегося непосредственно к сути тест-сценария) и сделать тесты максимально простыми и "читаемыми" для любого
разработчика.
